Help

Instructions: *** Works best in the Google Chrome Browser.***
  • Drag to Pan
  • Mouse Wheel to Zoom
  • Trace points are draggable (trace).
  • Right click to remove trace points.
  • Type in an h when finished creating a trigonometry function to turn it into it's hyperbolic form
  • Type (3,2) to plot a point (movable).
  • Type line AB to graph a line through points A and B.
  • Type segment AB to graph line segment AB.
  • Type sector ABC to graph a sector of a circle with center A, and endpoints B and C
  • Type circle AB to graph a circle with center A through B.
  • Type parabola ABC with focus A and line through B and C.
  • Type ellipse ABC with foci A and B through C.
  • Type hyperbola ABC with foci A and B through C.
  • Type lagrange ABCDE for Lagrange Polynomial through pts.
  • Type triangle ABC with points A, B, and C.
  • Type x = 3 for draggable vertical line (VLT).
  • Conic Sections through points are dynamic.
  • Conic Sections are constrained using trace points.
  • Type f(x)= to define functions. Note: You can type any name with (x) at the end of it to make it into a defined function
  • Type f(3) to evaluate functions.
  • Calculus operations are dynamic as well.
  • Hit refresh to start over. :)
  • More capabilities coming soon.

Open Graphing Calculator is an open source project, feel free to fork the source on GitHub, submit issues, and contribute.
